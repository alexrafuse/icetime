## Curling Club Rental Management System

Project Requirements

Overview: Rental management system for curling clubs

Frontend
- Filament PHP 3+

Backend
- Laravel 11 PHP 8.3+

Club Specifics
 - Four sheets of ice
 - Front bar area
 - Front kitchen
 - Back kitchen

Requirements
- Ability to book areas of the club for events
- Ability to set availability for each area
- Ability to set pricing for each area
- Ability to set up recurring events
- Ability to set up one-time events
- Ability to send notifications to users when their event is confirmed
- Ability to send notifications to users when their event is cancelled
- Ability to send notifications to users when their event is rescheduled
- Ability to export events to a CSV file

Testing
- Use PHPUnit for unit testing.
- Write tests for all new features and functionality.
- Follow the TDD (Test-Driven Development) approach.
